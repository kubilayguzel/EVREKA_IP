Yardımcı Eklenti Kurulumu Gerekli

TÜRKPATENT sitesinden anlık ve otomatik sorgulama yapabilmek için tarayıcınıza "Evreka IP Sorgu Yardımcısı" eklentisini güvenle kurmanız gerekmektedir.

Eklenti Yakında Chrome Web Mağazası'nda
Geliştirme Aşamasında: Bu eklenti henüz Chrome Web Mağazası'nda yayınlanmamıştır. Şu anda geliştirme sürecinde olup, yakında kullanıcılarımızın hizmetine sunulacaktır.
Chrome Web Mağazası'ndan Güvenle Yükle
  • Eklenti, sadece TÜRKPATENT sitesinde çalışmak üzere tasarlanmıştır.
  • Kurulum tek tıkla tamamlanır ve güncellemeler otomatiktir.
  • Kişisel verilerinize veya tarama geçmişinize asla erişmez.
